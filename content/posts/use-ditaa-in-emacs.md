-   ditaa 可以繪製對應的矩形圖,如下

```org
#+BEGIN_SRC ditaa :file ../images/linux02.png :cmdline -E -r -s 1.0
+---------------------------------------+
|                Applications           |
|                cRED                   |
+-------------------+                   |
|    cBLU           |                   |
|    Libraries      |                   |
+-------------------+-------------------+
|                                       |
|   Kernel          +-------------------+
|   cYEL            |           cGRE    |
|           +-------+--------+  Drivers |
|           |   Firmware c1AF|          |
+-----------+----------------+----------+
|               Hardware  c1AC          |
+---------------------------------------+
```

\#+END\_SRC
