* 设置 Emacs 的默认编码格式
#+BEGIN_SRC emacs
(prefer-coding-system 'utf-8-unix)
#+END_SRC

* 改变文件编码格式

- C-x C-m f utf-8-unix RET 将当前文件转换为 utf-8 编码
- C-x C-m c RET C-x C-w RET 另存为指定编码
